1앱 개요

English Lesson Generator는 유튜브 영상 링크를 입력하면 AI가 자동으로 영어 수업 워크시트를 생성해주는 도구입니다. 별도 설치 없이 브라우저에서 HTML 파일을 열어 바로 사용할 수 있습니다.

전체 흐름
유튜브 링크 입력 → 영상 분석 → 추가 영상 검색 (선택) → AI 수업자료 생성 → PDF/HTML 저장

생성되는 수업자료 구성:

💡Key Expressions핵심 표현 6–8개 + 해설
📖Vocabulary어휘 8–10개 + 품사 + 한국어 뜻
❓Comprehension Questions내용 이해 확인 문제
💬Discussion Questions토론 주제 질문
🗒️Teacher Notes교사용 참고 메모 (화면 전용, 인쇄 제외)

3API 키 발급 및 설정

앱을 사용하려면 YouTube API Key 1개와 AI API Key 1개가 필요합니다.

🔴 YouTube Data API v3 Key (공통 — v1/v2 모두 필요)
  1. Google Cloud Console (console.cloud.google.com)에 접속하여 프로젝트를 생성합니다.
  2. 좌측 메뉴 APIs & Services → Library에서 YouTube Data API v3를 검색 후 활성화합니다.
  3. APIs & Services → Credentials → + CREATE CREDENTIALS → API key를 클릭합니다.
  4. 생성된 키(AIza...)를 복사합니다.
YouTube API는 하루 10,000 유닛 무료입니다. 영상 분석 1회 = 약 3유닛, 검색 1회 = 약 100유닛 사용됩니다.
🟣 Anthropic Claude API Key (v1 전용)
  1. Anthropic Console (console.anthropic.com/keys)에 로그인합니다.
  2. API Keys → Create Key를 클릭합니다.
  3. 생성된 키(sk-ant-...)를 복사합니다.
🔵 Google Gemini API Key (v2 전용)

발급받은 Gemini API 키(AIza...)를 Google Gemini API Key 입력란에 붙여넣습니다.

API 키는 화면에 입력 후 자동 저장되지 않습니다. 브라우저를 닫으면 재입력이 필요합니다. 키를 외부에 공유하지 마세요.
4Step 1 — 기본 영상 선택
입력
유튜브 영상 URL을 붙여넣고 🔎 영상 분석 버튼을 클릭합니다.

지원하는 URL 형식:

  • https://www.youtube.com/watch?v=VIDEO_ID
  • https://youtu.be/VIDEO_ID

분석이 완료되면 📌 기본 영상 슬롯에 썸네일·제목·채널명·조회수가 표시됩니다. 기본 영상은 항상 수업자료에 포함됩니다.

자막(transcript)이 있는 영상일수록 AI가 더 정확한 수업자료를 생성합니다. 자동 생성 자막도 활용됩니다.
5Step 2 — 추가 영상 검색
입력
수업 주제 키워드를 영어로 입력하고 기간을 선택한 뒤 🔍 검색 버튼을 클릭합니다.

검색 옵션:

  • 키워드 — 기본 영상의 제목이 자동으로 채워집니다. 필요에 따라 수정하세요.
    예: climate change impacts, AI in daily life
  • 기간 필터 — 최근 1년 / 최근 2년 / 전체 중 선택

검색 후 조회수 1위 영상이 🔍 추가 영상 슬롯에 자동 배치됩니다.
마음에 들지 않으면 하단에 표시되는 검색 결과 목록에서 원하는 영상을 클릭해 교체할 수 있습니다.

교체 방법
추가 영상 슬롯의 🔄 다른 영상으로 교체 버튼을 누르면 교체 모드가 활성화되고, 하단 검색 결과에서 원하는 영상을 클릭하면 즉시 교체됩니다.
추가 영상 검색은 선택 사항입니다. 기본 영상만으로도 수업자료 생성이 가능합니다.
6Step 3 — 수업자료 생성

영상 구성이 완료되면 아래 설정을 입력하고 ✨ 수업자료 생성 버튼을 클릭합니다.

설정 항목
항목설명예시
학습자 수준 AI가 어휘·표현 난이도를 맞춥니다 Beginner / Intermediate / Advanced
수업 날짜 워크시트 상단에 표시됩니다 2025-03-17
학급명 워크시트 상단에 표시됩니다 Advanced Class B
수업 주제 AI 프롬프트에 반영되어 콘텐츠 방향을 잡습니다 Climate Change & Solutions

생성 중에는 진행 상황이 [1/2] "영상 제목..." 수업자료 생성 중... 형태로 표시됩니다.
영상 1개당 약 15–30초가 소요됩니다.

생성 도중 창을 닫거나 새로고침하면 처음부터 다시 시작해야 합니다.
7생성 결과물 구성

영상당 워크시트 1장이 생성됩니다. 각 워크시트는 다음 섹션으로 구성됩니다.

📋 Content Summary
영상 내용의 2–3문장 요약. 영어 원문과 한국어 번역이 함께 제공됩니다.
💡 Key Expressions
학습자 수준에 맞는 핵심 표현 6–8개. 각 표현의 한국어 설명 및 예문이 포함됩니다.
📖 Vocabulary
주요 어휘 8–10개. 단어·품사·한국어 뜻이 카드 형태로 표시됩니다.
❓ Comprehension Questions
영상 내용을 확인하는 이해도 점검 문제입니다.
💬 Discussion Questions
영상 주제를 바탕으로 한 토론·발화 유도 질문입니다.
🗒️ Teacher Notes (화면 전용)
교사를 위한 참고 메모입니다. 인쇄·PDF에는 포함되지 않습니다.
8PDF 출력 및 HTML 저장
🖨️ PDF 출력 / 인쇄
브라우저 인쇄 다이얼로그를 엽니다. 대상 프린터를 "PDF로 저장"으로 선택하면 PDF 파일로 저장됩니다. A4 세로 방향을 권장합니다.
💾 HTML로 저장
결과 워크시트를 독립된 HTML 파일로 저장합니다. 인터넷 없이도 브라우저에서 열람·인쇄할 수 있어 배포용으로 적합합니다.
↩️ 새 수업 만들기
모든 입력값과 결과를 초기화하고 처음 상태로 돌아갑니다.
PDF 출력 시 브라우저 설정에서 배경 그래픽 포함을 체크하면 색상과 배경이 그대로 인쇄됩니다.
9사용 팁 및 주의사항
✅ 좋은 결과를 위한 팁
  • 자막 있는 영상 선택 — 영어 자막(CC)이 있는 영상을 사용하면 AI가 실제 내용을 기반으로 더 정확한 수업자료를 만듭니다.
  • 수업 주제를 구체적으로 입력 — "environment" 보다 "plastic pollution solutions"처럼 구체적인 키워드가 효과적입니다.
  • 수준에 맞는 영상 선택 — Beginner 수준이라면 느리고 명확하게 말하는 영상을, Advanced라면 네이티브 속도의 영상을 선택하세요.
  • 추가 영상 교체 활용 — 자동 배치된 영상이 맞지 않으면 검색 결과에서 주제에 더 잘 맞는 영상으로 교체하세요.
  • 학급명·날짜 미리 입력 — 워크시트 상단에 바로 출력되므로 수업 직전에 입력해두면 편리합니다.
⚠️ 주의사항
  • API 키는 브라우저 세션이 끝나면 사라집니다. 매번 재입력하거나 메모장에 별도 보관하세요.
  • YouTube API 무료 할당량(일 10,000유닛)을 초과하면 다음 날 UTC 기준으로 초기화될 때까지 검색이 불가합니다.
  • 생성된 수업자료는 AI가 만든 것으로, 교사가 내용을 검토한 후 사용하는 것을 권장합니다.
  • 자막이 없는 영상은 영상 제목·설명을 기반으로 수업자료가 생성되므로 정확도가 낮을 수 있습니다.
